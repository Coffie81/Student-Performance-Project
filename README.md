# Student Performance Data Analysis

## Project Overview
This project analyzes student exam performance to identify weak subjects, top-performing students, and performance differences by gender.

## Dataset
The dataset contains:
- Student ID
- Gender
- Math, Reading, and Writing scores
- Attendance percentage

## Tools Used
- Python
- Pandas
- Matplotlib

## Analysis Performed
- Calculated average score for each subject
- Identified the hardest subject based on lowest average score
- Found top-performing students
- Compared performance by gender
- Created visualizations

## Key Insights
- Mathematics was identified as the hardest subject.
- Female students performed better in writing.
- Students with balanced performance achieved higher overall scores.

## Recommendation
Provide additional academic support and targeted interventions in Mathematics to improve student outcomes.
